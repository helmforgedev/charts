============================================================
  GETTING STARTED
============================================================

1. Wait for the workload to be ready:
   kubectl wait --for=condition=ready pod -l app.kubernetes.io/name=flowise -n {{ .Release.Namespace }} --timeout=300s

2. Validate the API responds (port-forward from ACCESS INFORMATION first):
   curl -sf http://localhost:3000/api/v1/ping && echo " <- Flowise API OK"

3. Open the Web UI, create your first chatflow, and add your model provider
   credentials (OpenAI, Azure, local models, ...).
